# 💻 ローカルセットアップガイド

このブログを自分のパソコンで動かすための手順を説明します。

---

## 📋 必要条件

- **Node.js**: v20.0.0 以上
- **Git**: 最新版
- **pnpm**: 推奨（`npm install -g pnpm` でインストール可能）

---

## 🚀 セットアップ手順

### 1. リポジトリのクローン

```bash
git clone <あなたのリポジトリURL>
cd Svelte-site-blog
```

### 2. 自動セットアップの実行

プロジェクトルートにある `setup.sh` を使用して、依存関係のインストールと初期設定を自動で行います。

```bash
bash setup.sh
```

このスクリプトは以下の処理を行います：

- ライブラリのインストール (`pnpm install`)
- `.env` ファイルの作成（存在しない場合）
- アップロード・バックアップ用ディレクトリの作成

### 3. 開発サーバーの起動

```bash
pnpm run dev
```

起動後、ターミナルに表示される URL（通常は `http://localhost:5173`）にアクセスしてください。

---

## 🔑 初回管理者登録

サイトを初めて起動してアクセスすると、自動的に「管理者セットアップ画面」が表示されます。

- 任意の **ユーザー名** と **パスワード** を設定してください。
- 登録が完了すると、自動的にダッシュボードへログインします。

---

## 📁 主要なディレクトリ構造

- `src/routes`: ページのルーティング
  - `(admin)/dashboard`: 管理画面
  - `(auth)`: ログイン・会員登録
  - `(pages)`: 公開ブログページ
- `src/lib`: 共通ライブラリ・コンポーネント
- `static/uploads`: アップロードされた画像（ローカル保存設定時）
- `blog.db`: データベースファイル（ローカル保存時、自動生成）

`http://localhost:5173` にアクセスし、初期セットアップ画面が表示されれば成功です！
