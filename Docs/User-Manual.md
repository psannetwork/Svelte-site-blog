# ✍️ 記事執筆・管理マニュアル

このブログの心臓部である Editor.js を使いこなし、美しい記事を作成するためのマニュアルです。

---

## 📝 執筆の基本 (Editor.js)

Editor.js は「ブロック形式」のエディタです。テキスト、画像、見出しなどを「ブロック」として追加していきます。

### ブロックの追加

- 新しい行で **[+]** ボタンを押すか、キーボードの `/` (スラッシュ) を入力すると、追加できるブロックのメニューが出ます。

### 便利なブロック一覧

1.  **見出し (Header)**: 記事の構造を作ります。
2.  **画像 (Image)**:
    - 画像をドラッグ＆ドロップで追加。
    - 画像をクリックすると、**「角をドラッグ」してサイズを自由に調整**できます。
3.  **リスト (List)**: 箇条書きや番号付きリスト。
4.  **引用 (Quote)**: 重要な言葉を目立たせます。
5.  **コード (Code)**: プログラムのコードを表示します。
6.  **区切り線 (Delimiter)**: 内容の区切りにおしゃれな線を入手。

---

## ⌨️ ショートカットキー

- **Ctrl + S**: 執筆中の内容を**その場で保存**します（下書き保存にも便利）。
- **Ctrl + Z**: 間違えて消したときなどに、一つ前に戻します。
- **Tab**: ツールメニューなどを素早く移動。

---

## 🖱️ 右クリックメニュー

エディタ内の任意のブロックを**右クリック**すると、そのブロック専用のメニューが開き、以下の操作が瞬時に行えます。

- **削除**: ブロックを消します。
- **上に移動 / 下に移動**: 順番を入れ替えます。
- **複製**: ブロックをコピーします。

---

## 💬 コメントの管理

ダッシュボードの **[Comments]** から、読者から届いたコメントを確認・削除できます。

- 悪質なコメントは管理者権限で即座に削除可能です。
- 返信があった場合、あなたのベルマークアイコンに通知が届きます。
