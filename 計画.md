# 🚀 Svelteフル機能ブログプロジェクト計画まとめ

## 1️⃣ セットアップガイド (開発者用)

本プロジェクトを開始するための手順です。

### 1. プロジェクトの初期化
```bash
# 依存関係のインストール
npm install

# 必要なライブラリの追加 (DB, Auth等)
npm install better-sqlite3 lucia @lucia-auth/adapter-sqlite
npm install -D @types/better-sqlite3 tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### 2. データベースの準備
`src/lib/server/db.ts` が自動的に `blog.db` を作成します。

### 3. 開発サーバーの起動
```bash
npm run dev -- --port 3000
```

## 5️⃣ Renderでの公開方法

### Dockerfile例

```dockerfile
FROM node:20-alpine AS build
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build

FROM node:20-alpine
WORKDIR /app
COPY --from=build /app .
EXPOSE 3000
CMD ["node", ".svelte-kit/output/server/index.js"]
```

### デプロイ手順

1. Renderにログイン → **New → Web Service**
2. GitHubリポジトリを指定
3. **Environment** → Docker
4. Renderが自動でビルド＆URL発行
5. プレビューとして誰でもアクセス可能
